# Excel Data Analysis Portfolio

Welcome to my Data Analytics Portfolio.

This repository serves as a central hub for my Excel-based data analysis projects, where I explore datasets, perform data transformations, and build interactive dashboards to uncover meaningful insights.

These projects demonstrate the typical workflow of a data analyst:    

`Raw Data → Data Cleaning → Data Modeling → Analysis → Dashboard → Insights`   

By applying this process, I aim to extract insights that support better decision-making and data storytelling.

## 🚀 Tools & Skills Demonstrated

The projects in this portfolio utilize the following tools and techniques in Microsoft Excel:

Power Query – Data extraction, cleaning, and transformation

Power Pivot – Data modeling and table relationships

Pivot Tables & Pivot Charts – Data summarization and visualization

DAX (Data Analysis Expressions) – Custom measures and calculations

Interactive Dashboards – User-friendly data exploration

These tools allow Excel to operate similarly to business intelligence platforms such as Microsoft Power BI.
