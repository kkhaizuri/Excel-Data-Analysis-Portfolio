# Excel Data Analysis Portfolio

## Introduction

Welcome to my Excel Data Analytics Portfolio. This repository showcases projects where I use data analysis and visualization techniques to transform raw datasets into meaningful insights.  

Through these projects, I demonstrate how Microsoft Excel can function as a powerful data analytics tool, using features such as data transformation, data modeling, and interactive dashboards.

These projects reflect the typical workflow of a data analyst:   
' Raw Data → Data Cleaning → Data Modeling → Analysis → Dashboard → Insights '   
By applying this process, I aim to extract insights that support better decision-making and data storytelling.
